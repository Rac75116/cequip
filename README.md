# cequip

## Build

### Prerequisites

- uv
- CMake
- Ninja
- clang++ (Linux/MacOS)
- msvc (Windows)

### Build

Linux/MacOS:
- `./scripts/configure.sh`
- `./scripts/build.sh`

Windows:
- `.\scripts\configure.bat`
- `.\scripts\build.bat`

### Install

Linux/MacOS:
- `./scripts/install.sh`

Windows:
- `.\scripts\install.bat`
